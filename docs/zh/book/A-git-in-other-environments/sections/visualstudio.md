### Visual Studio 中的 Git

从 Visual Studio 2019 16.8 版本开始，Visual Studio 将 Git 工具直接内置到
IDE 中。

工具支持以下 Git 功能：

- 创建与克隆仓库。

- 打开与浏览仓库的历史。

- 创建与检出分支与标签。

- 储藏、暂存与提交改动。

- 抓取、拉取、推送或同步提交。

- 合并与变基分支。

- 解决合并冲突。

- 查看差异。

- …​ 等等。

阅读 [official
documentation](https://docs.microsoft.com/visualstudio/version-control)
以了解更多内容。
