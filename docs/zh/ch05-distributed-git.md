## 分布式 Git

你现在拥有了一个远程 Git
版本库，能为所有开发者共享代码提供服务，在一个本地工作流程下，你也已经熟悉了基本
Git 命令。你现在可以学习如何利用 Git 提供的一些分布式工作流程了。

这一章中，你将会学习如何作为贡献者或整合者，在一个分布式协作的环境中使用
Git。
你会学习为一个项目成功地贡献代码，并接触一些最佳实践方式，让你和项目的维护者能轻松地完成这个过程。另外，你也会学到如何管理有很多开发者提交贡献的项目。
